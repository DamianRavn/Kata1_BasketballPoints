# Kata1_BasketballPoints
You are counting points for a basketball game, given the amount of 2-pointers scored and 3-pointers scored, find the final points for the team and return that value.

Examples

points(1, 1) ➞ 5

 

points(7, 5) ➞ 29

 

points(38, 8) ➞ 100

 

points(0, 1) ➞ 3

 

points(0, 0) ➞ 0
